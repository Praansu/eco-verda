# EcoVerda

A simple e-commerce website for eco-friendly products. Built as a frontend learning project using HTML, CSS, and JavaScript.

## What It Is
- Multi-page product catalog
- About Us, Blog, and Product pages
- Responsive design that works on mobile and desktop
- Team portfolio pages

## What I Learned
- HTML5 semantic markup
- CSS layout and responsive design
- JavaScript for interactive features
- Web project structure and organization
